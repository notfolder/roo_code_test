# 備品管理システム

会社のオフィス備品（PC・プロジェクター等）の貸出状況をリアルタイムで管理するWebアプリです。

## 前提条件

- Docker
- Docker Compose

## 起動方法

```bash
docker compose up --build
```

起動後、ブラウザで http://localhost:8080 にアクセスしてください。

## 初期ログイン情報

| ログインID | パスワード | ロール |
|-----------|-----------|--------|
| admin | admin | 総務担当者 |

初回ログイン後、ユーザー管理画面からパスワードを変更することを推奨します。

## 操作説明

### 総務担当者（adminロール）

- 備品一覧：備品の在庫状況・借用者を確認できます
- 備品登録：管理番号と備品名を入力して備品を登録します
- 備品編集：備品名を変更します（管理番号は変更不可）
- 備品削除：在庫ありの備品のみ削除できます（貸出中は削除不可）
- 貸出登録：借用者名を入力して貸出を記録します
- 返却登録：貸出中の備品を返却済みに変更します
- ユーザー管理：ユーザーの登録・編集・削除ができます

### 一般社員（staffロール）

- 備品一覧：備品の在庫状況・借用者を閲覧できます（操作不可）

## データの永続化

PostgreSQLのデータはDockerボリューム（postgres_data）に永続化されます。
データを初期化する場合は以下を実行してください。

```bash
docker compose down -v
docker compose up --build
```
