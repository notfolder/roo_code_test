# 正規 Q-Q プロット可視化アプリ

## 概要
CSV ファイルに含まれる 2 つの数値カラムの分布形状を正規 Q-Q プロットで比較表示する Web アプリケーション。

## 依存ライブラリのインストール
```bash
pip install -r requirements.txt
```

## アプリの起動方法
```bash
streamlit run app.py
```

## 使用方法
1. アプリを起動後、CSV ファイルをアップロードしてください。
2. 数値カラムが自動的に検出され、ドロップダウンに表示されます。
3. 2 つの異なる数値カラムを選択してください。
4. 正規 Q-Q プロットと基本統計量が自動的に表示されます。

## CSV ファイルの形式
- 数値カラム（int または float）を含む CSV ファイル
- ヘッダー行があること
- データ数は最小 3 件以上

## 例
```csv
id,value_a,value_b
1,120.5,45.2
2,135.8,52.1
3,110.2,48.7
```

## 機能一覧
- CSV ファイルのアップロード
- 数値カラムの自動検出と選択
- 正規 Q-Q プロットの生成（標本分位数 vs 理論分位数）
- 45 度参照線の表示
- 基本統計量（平均、標準偏差、最小値、最大値、データ数）の表示

## エラーハンドリング
- 数値カラムがない場合：エラーメッセージを表示
- 標準偏差が 0 の場合：変化するデータがありませんと表示
- データ数が 3 件未満の場合：十分なデータ数がない旨を表示
- ファイル読み込みエラー：エラーメッセージを表示

## 技術スタック
- Python 3.10+
- Streamlit
- pandas, numpy, matplotlib, scipy
