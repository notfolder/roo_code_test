## バリデーションタスク詳細

### T01 依存ライブラリの設定
- **目的**: アプリが正常に動作するための依存関係を定義
- **手順**: requirements.txt に必要なライブラリを記載

### T02 CSV ファイル読み込み機能の実装
- **目的**: ユーザーから CSV ファイルを受け取り、DataFrame に変換
- **手順**: Streamlit の file_uploader と pandas の read_csv を使用

### T03 統計量計算機能の実装
- **目的**: 数値カラムの平均、標準偏差、最小値、最大値を計算
- **手順**: pandas の describe() または numpy の関数を使用

### T04 正規 Q-Q プロット描画機能の実装
- **目的**: 標本分位数 vs 理論分位数の散布図を 45 度参照線付きで描画
- **手順**: matplotlib の scatter() と直線描画を使用

### T05 理論分位数計算機能の実装
- **目的**: scipy.stats.norm.ppf で正規分布の理論分位数を計算
- **手順**: np.arange() で分位数インデックスを生成し、ppf() を適用

### T06 基本統計量テーブル表示機能の実装
- **目的**: カラムの統計量を表形式で表示
- **手順**: pandas の DataFrame を Streamlit の table() で表示

### T07 エラーハンドリングの実装
- **目的**: 異常系で適切なメッセージを表示
- **手順**: Streamlit の error() または warning() を使用

### T08 README.md の作成
- **目的**: 起動方法、使用方法を文書化
- **手順**: プロジェクトルートに README.md を作成

### T09 最終確認タスク
- **目的**: 要件定義書と詳細設計書のすべての項目が満たされているか確認
- **手順**: すべてのタスクを完了後、レビューを実施
